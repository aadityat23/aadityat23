## Featured Projects

### 1. End-to-End Data Engineering Pipeline
> Automated data infrastructure from source ingestion to analytical output — built for scale.

Designed and implemented a complete ETL pipeline that ingests raw data from external APIs and flat files, applies multi-stage transformations, loads into a structured SQL data warehouse, and surfaces insights via automated reporting. Architected with modularity and fault tolerance in mind.

**Stack:** `Python` `SQL` `REST APIs` `Pandas` `PostgreSQL` `Apache Airflow` `Docker`  
**Impact:** Reduced manual reporting effort by automating the full data lifecycle; schema designed to support 10x data volume growth without structural changes.

**Stack:** `Python` `Pandas` `Power BI` `SQL` `DAX`  
**Impact:** Enables data-driven scouting and tactical decisions; demonstrates applied BI skills across the full pipeline — from raw CSV to executive-ready dashboard.
